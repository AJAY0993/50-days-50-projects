# Clock App README

This is a simple HTML, CSS, and JavaScript-based clock application that displays the current time, date, and includes a toggle switch for changing between light and dark modes.

## Table of Contents

- [Features](#features)
- [Installation](#installation)
- [Usage](#usage)
- [File Structure](#file-structure)
- [Credits](#credits)
- [License](#license)

## Features

1. **Real-time Clock Display:** The application provides a real-time clock display with hour, minute, and second hands that move in real-time.

2. **Date and Day Display:** In addition to the time, the application also displays the current day of the week, month, and date.

3. **Dark Mode Toggle:** Users can switch between light and dark modes by clicking the "Dark mode" button, which changes the color scheme of the clock and background.

## Installation

To run this clock application locally, follow these steps:

1. Clone the repository to your local machine:

   ```shell
   git clone https://github.com/AJAY0993/50-days-50-projects/tree/main/Clock
   ```

2. Open the `index.html` file in your web browser.

## Usage

1. Open the application in your web browser.

2. The clock will start displaying the current time, day, month, and date.

3. To switch between light and dark modes, click the "Dark mode" button.

## File Structure

- **index.html:** The main HTML file that contains the structure of the clock application and links to external CSS and JavaScript files.

- **normalize.css:** A CSS file that normalizes the default styles across different web browsers.

- **style.css:** The CSS file that defines the styles for the clock, dark mode, and layout.

- **main.js:** The JavaScript file that handles the clock's functionality, including real-time updates and dark mode toggle.
